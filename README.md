# Loja Virtual

Esse é um projeto criado usando T3 stack.
Node v18 (18.16.0)

### Tecnologias utilizadas:

> - Next
> - TRPC
> - Tailwind
> - Next Auth
> - Mantine
> - Tabler Icons

### Inicializando o Projeto:

Rode o comando abaixo para instalar as dependências do projeto:

```bash
pnpm install
```

Rode o comando abaixo para iniciar o projeto:

```bash
pnpm dev
```
